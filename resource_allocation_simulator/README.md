# Project 3: Resource Allocation Simulator

## 🚀 Overview
This project is an algorithmic Python simulation designed to model and optimize how computing resources are distributed across various tasks or environments. In the cloud-native ecosystem, resource management directly impacts system uptime, cost-efficiency, and resilience—making this project highly relevant to Infrastructure as Code (IaC) and system operations in a DevSecOps model.

## 🛠️ Features & Methodology
* **Dynamic Simulation:** Models concurrent processes requesting resources (CPU/RAM) and evaluates scheduling strategies.
* **Bottleneck Analysis:** Identifies deadlocks, resource starvation points, and system overloads through data logs.
* **Optimization Algorithms:** Implements logic to maximize efficiency, minimizing waste and system idle time.
