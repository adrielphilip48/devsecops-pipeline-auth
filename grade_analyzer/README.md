# Project 1: Student Grade Analyzer

## 🚀 Overview
This project is an automated Python utility designed to parse complex educational datasets, compute key statistical metrics, and output visual data trends. From a DevSecOps engineering perspective, this project establishes my core capabilities in data parsing, scripting, log generation, and automated reporting—all crucial for tracking system metrics and telemetry data.

## 🛠️ Features & Methodology
* **Automated Data Ingestion:** Reads and validates structured `student_grades.csv` data inputs.
* **Telemetry & Report Generation:** Compiles data aggregates into automated flat-text reports (`report.txt`).
* **Visual Data Analytics:** Utilizes data visualization libraries to output trend charts (`.png`), mirroring system performance dashboards.

## 🎯 Strategic Alignment
Before securing a pipeline, a DevSecOps engineer must understand how to handle, parse, and monitor data flows. This application demonstrates my foundational fluency in Python backend scripting and data hygiene, preparing me for complex infrastructure scripting at **Braeburn International School Arusha**.
